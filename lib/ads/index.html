Publicité par Minteck Projects

Cet API fournit une publicité dynamique pour les services de Minteck Projects. Il est utilisé partout où les services de Minteck Projects contiennent de la publicité.

Méthodes d'accès

static

Publicités statiques — Ces publicités ne sont pas générées automatiquement et ne fournissent qu'une seule publicité. Son utilisation est fortement déconseillée.

Arguments

  • /lib/ads/static — URL racine
  • /lib/ads/static/fr — Publicités en français
  • /lib/ads/static/en — Publicités en anglais
  • /lib/ads/static/fr/banner — Publicités sous forme de bannière en français
  • /lib/ads/static/fr/banner/4 — Publicité n°4 sous forme de bannière en français

L'URL doit être sous la forme /lib/ads/static/<Langue>/<Format>/<Identifiant>

source

Fichiers d'origine des publicités — Ces publicités sont les fichiers au format GIMP des publicités dans le dossier /static. Son utilisation est fortement déconseillée.

Arguments

  • /lib/ads/source — URL racine
  • /lib/ads/source/fr-*.xcf — Publicités en français
  • /lib/ads/source/en-*.xcf — Publicités en anglais
  • /lib/ads/source/fr-banner-*.xcf — Publicités sous forme de bannière en français
  • /lib/ads/source/fr-banner-4.xcf — Publicité n°4 sous forme de bannière en français

L'URL doit être sous la forme /lib/ads/static/<Langue>/<Format>/<Identifiant>

dynamics

Publicités dynamiques — Ces publicités sont générées automatiquement parmi toutes les publicités possibles.

Arguments

  • lang — (obligatoire) Langue de la publicité
  • format — (optionnel) Format de la publicité, banner par défaut

Dimensions des publicités

Si vous êtes intéressé pour créer vos propres publicités, vous pouvez suivre les dimensions suivantes et nous les envoyer par mail à minteck.projects+adrequests@gmail.com

  • Bannière :
    • Pixel Perfect : 700x200
    • Basse résolution : 350x100
    • Ratio : 7:2
  • Côté :
    • Pixel Perfect : 700x500
    • Basse résolution : 350x250
    • Ratio : 7:5
  • Bannière fine :
    • Pixel Perfect : 700x150
    • Basse résolution : 350x75
    • Ratio : 7:1,5

L'URL doit être sous la forme /lib/ads/dynamics/?lang=<Langue>&format=<Format>
